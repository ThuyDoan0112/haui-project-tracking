# `haui-project-tracking`

## `Setup`

```bash
# install the dependencies
npm install
```

```bash
# create environment variables
cp .env.example .env
```

## `Development Server`

```bash
# start the development server on http://localhost:3000:
npm run dev
```

## `Production`

```bash
# build the application for production
npm run build
```

```bash
# locally preview production build
npm run preview
```
